I used "float" style to controls bloks position.

And I used a new fashion "display:flex" style in "nav" section. It works very well!

ul {
    height: 70px;
    width: 80%;
    margin: auto;
    display: flex;
    flex-direction: row;
    justify-content: space-between;
}

"background-size:contain" style is useful for adjusting the image to fit the screen size.

#big_view {
    position: fixed;
    top: 0;
    left: 0;
    z-index: 20;
    width: 100vw;
    height: 100vh;
    background-image: url('img/load.png');
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center;
    background-color: #000;
    display: none;
    cursor: pointer;
}

Java script monitored browsers events.

$(document).on("scroll", function () {
        if (bussyScroll) {
            return;
        }
        bussyScroll = true;
        setTimeout(function () {
            onScroll();
            bussyScroll = false;
        }, 100);
});
$(window).on("resize", function () {
        if (bussyResize) {
            return;
        }
        bussyResize = true;
